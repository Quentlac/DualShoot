Téléchargement

Téléchargez le fichier compressé puis décompressez le grâce à un
logiciel de décompression comme WinRar. Executer
ensuite editeur_map.exe./!\ Requiert une version récente de Java /!\

Ensuite, une fois votre map finis appuyez
sur ESPACE et un fichier map sera créé.
Il se trouve dans le même dossier que editeur_map.exe.
